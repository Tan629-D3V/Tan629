A sophisticated real-time NLP chatbot that leverages the power of OpenAI's GPT models to provide intelligent, context-aware conversations with users.

## Project Overview

Synapse is a full-stack AI chatbot application that demonstrates modern web development practices combined with cutting-edge AI technology. The project showcases real-time communication, secure authentication, and intelligent response generation.

## Technical Architecture

### Frontend (React.js)
- **State Management**: React Context API for global state
- **UI Framework**: Chakra UI for consistent, accessible components
- **Real-time Communication**: Socket.io client for live chat
- **Routing**: React Router for navigation
- **HTTP Client**: Axios for API requests

### Backend (Node.js + Express)
- **Authentication**: JWT tokens with refresh mechanism
- **Database**: MongoDB with Mongoose ODM
- **Real-time**: Socket.io server for WebSocket connections
- **AI Integration**: OpenAI API with request optimization
- **Security**: Rate limiting, input validation, CORS

### Database Schema
```javascript
// User Schema
{
  username: String,
  email: String,
  password: String (hashed),
  createdAt: Date,
  lastActive: Date
}

// Message Schema
{
  userId: ObjectId,
  content: String,
  role: String, // 'user' or 'assistant'
  timestamp: Date,
  conversationId: String
}
```

## Installation & Setup

<InfoAlert title="Prerequisites" type="info">
  Make sure you have Node.js (v16+) and MongoDB installed on your system.
</InfoAlert>

### 1. Clone the Repository
```bash
git clone https://github.com/Tau629-D3V/synapse-chatbot.git
cd synapse-chatbot
```

### 2. Install Dependencies
```bash
# Install backend dependencies
cd backend
npm install

# Install frontend dependencies
cd ../frontend
npm install
```

### 3. Environment Configuration
Create `.env` files in both backend and frontend directories:

**Backend (.env)**
```env
MONGODB_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret_key
OPENAI_API_KEY=your_openai_api_key
PORT=5000
```

**Frontend (.env)**
```env
REACT_APP_API_URL=http://localhost:5000
REACT_APP_SOCKET_URL=http://localhost:5000
```

### 4. Run the Application
```bash
# Start backend server
cd backend
npm run dev

# Start frontend development server
cd ../frontend
npm start
```

## API Endpoints

### Authentication
- `POST /api/auth/register` - User registration
- `POST /api/auth/login` - User login
- `POST /api/auth/refresh` - Refresh JWT token
- `POST /api/auth/logout` - User logout

### Chat
- `GET /api/chat/history` - Get chat history
- `POST /api/chat/message` - Send message to AI
- `DELETE /api/chat/history` - Clear chat history

### WebSocket Events
- `join` - Join chat room
- `message` - Send/receive messages
- `typing` - Typing indicators
- `disconnect` - Handle disconnection
